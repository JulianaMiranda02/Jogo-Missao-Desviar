 Missão Desviar

Jogo desenvolvido em Python utilizando a biblioteca Pygame.

🎮 Sobre o Jogo

Missão Desviar é um jogo de reflexo e sobrevivência.

O jogador controla um personagem que precisa desviar de obstáculos que caem do topo da tela.
Quanto mais tempo você sobrevive, maior fica sua pontuação!
